# 🚀 Nobre Backend | API de Serviços

Este é o core backend do projeto **Nobre**, desenvolvido em Node.js. A arquitetura foi desenhada seguindo padrões de separação de responsabilidades (MVC/Services) para facilitar a manutenção e escalabilidade.

## 🛠️ Tecnologias Principais

* **Node.js** - Ambiente de execução.
* **Express** - Framework para rotas e middlewares.
* **Nodemailer** - Serviço para disparo de e-mails.

## 📁 Arquitetura do Sistema

```text
├── controllers/    # Lógica de controle e resposta das requisições
├── routes/         # Definição dos endpoints da API
├── services/       # Regras de negócio e integrações (E-mail, Contratos)
├── .env.example    # Modelo de configuração local
├── .gitignore      # Arquivos ignorados pelo Git
├── index.js        # Inicialização do servidor
├── server.js       # Configurações de porta e middlewares
└── package.json    # Dependências e scripts
````
